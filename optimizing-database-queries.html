Optimizing Database Queries in Large ERP Systems

One of the biggest challenges in the ERP project was slow reporting queries during peak admission season.

Techniques That Worked

  • Proper indexing on frequently filtered columns
  • Using Dapper for complex reporting instead of EF Core
  • Caching frequently accessed data with Redis (planned next)
  • Implementing pagination everywhere

Result: Reduced average query time from **8 seconds** to **under 300ms**.
